EsportsHelper Config Maker

Required fields
Riot Games account username
Riot Games account password
Optional
Time interval for each check in seconds (600 seconds by default). Each check time will fluctuate randomly between 0.8 and 1.5 times the time delay you set.
When set to True, the program will run in the background; otherwise, it will open a browser window (False by default).
Nickname for added account, "username" by default.
Mode selection: "safe" for Safe mode, "normal" for regular mode, the default is "safe." Check the GitHub page for more details.
Length of log brief information. Default is 10.
Leagues to watch, separated by commas (e.g., lcs, lla, lpl). Please note, names should be in lowercase.
Leagues to ignore, separated by commas (e.g., lms, cbol). Please note, names should be in lowercase.
Language selection for the program interface.
The maximum number of matches that can be watched at the same time, beyond which they will not be watched. Default is 4.
Negative value means always running; positive value is running by hours. Default -1.
Proxy address, not required for general users (e.g., "127.0.0.1:7890").
Discord webhook link for connector drops.
Whether or not you need to export the total drop details file, it will only be generated when the script is opened.
Operating system platform. Windows is set by default. If you want to use the program on Linux, please change the value here.
Enable support for using Chromium on Linux ARM64, requires platForm: "linux" and having the chromedriver in "/home/USERNAME/.local/share/undetected_chromedriver/chromedriver", more info: https://github.com/Yudaotor/EsportsHelper/wiki/The-Way-Using-Chromium-on-ARM64
Option of deleting video-player elements to save traffic (Risk).
Experimental feature to enable Desktop notifications.
Hibernation period, empty by default, allowed multiple periods. The format is "Start hour – End hour". Tabs will be closed and reopened at the end of sleep.
Option to ignore broadcasts.
Path to Chrome cookie files.
Chrome.exe location.
Option to monitor Drops number.
Toggle for auto-sleep mode: sleeping while no matches live, waking up when a match goes live (True by default).
Debug mode, default is False.
